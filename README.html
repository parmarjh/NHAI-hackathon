◈ ALGORITHM DISCLOSURE

The 3D Geometric Intelligence Pipeline

01

Stereo-Depth Estimation

The system uses monocular depth-mapping to project 2D camera pixels into a 3D coordinate space, allowing for volume estimation of potholes and physical height measurement of road signs.

02

Radiometric Analysis

Calculates the coefficient of retroreflection (Ra) by analyzing pixel intensity variations across multiple 3D angles, compensating for vehicle headlight scatter.

03

Neural Semantic Segmentation

Isolates road assets (Gantries, RPMs, Delineators) from environmental noise (dust, rain, foliage) using a deep convolutional backbone.

04

Cascading Inference

Queries the 4-tier API failover chain (Gemini/Claude) with the pre-processed visual tensors to generate high-fidelity IRC compliance scores.

◈ VISION & PURPOSE

What Problem Does This Solve?

Manual Measurement Risk

Traditional road auditing requires teams to exit vehicles and use handheld meters on high-speed expressways, posing extreme safety risks. Our AI/ML system performs 100% of measurements from inside a 100km/h vehicle.

Maintenance Lag

Infrastructure deterioration is often reported too late. Our 'Health Intelligence' engine detects potholes and cracks at the 'micro-failure' stage, reducing long-term NHAI repair costs by ~40%.

Human Subjectivity

Compliance auditing (IRC 67/35) is often inconsistent. Our neural network provides an objective, immutable score for every asset, logged with GPS and high-resolution visual evidence.

◈ SYSTEM DESIGN

How It Works

A single-file Python desktop application built on tkinter, Pillow, and the Anthropic SDK.

01

Image Ingestion

User browses or drops a road sign JPG/PNG/WEBP. Pillow loads and thumbnails it for the HUD preview. The raw bytes are Base64-encoded for the API payload.

02

Claude Vision API Call

The encoded image and a structured system prompt are sent to claude-opus-4-5 via the Anthropic Python SDK on a background thread, preventing UI freezing.

03

JSON Response Parsing

Claude returns a strict JSON object with 12 typed fields. The response is sanitised, parsed, and validated before being dispatched to the main thread via after().

04

AI/ML Determination

High-speed determination of reflectivity scores for Signs, RPMs, and Markings based on IRC 67/35 standards using convolutional neural network reasoning.

05

History & Export

Results are appended to a session list. The Compare tab auto-rebuilds. Export writes a formatted .txt or raw .json to disk via a save dialog.

None · Low · Moderate · High · Severe — surface wear classification.
